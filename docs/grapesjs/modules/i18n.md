# Internationalization (I18n)

GrapesJS allows for multi-language support and updating UI strings.

## Configuration

```javascript
const editor = grapesjs.init({
  // ...
  i18n: {
    locale: 'en',
    detectLocale: true,
    messages: {
      en: { /* ... */ },
      fr: { /* ... */ },
    }
  }
});
```

## Update Strings

Use the `I18n` API to update existing strings.

```javascript
editor.I18n.addMessages({
  en: {
    styleManager: {
      empty: 'Please select an element',
    }
  }
});
```

## Adding New Languages

Languages should follow ISO 639-1 standards. You can import community-provided locales or create your own.
