# Block Manager API

The Block Manager module handles blocks that can be dragged into the canvas.

## Methods

- `add(id, configuration)`: Add a new block.
- `get(id)`: Get a block by ID.
- `getAll()`: Get all blocks.
- `getAllVisible()`: Get all visible blocks.
- `remove(id)`: Remove a block.
- `getCategories()`: Get all categories.
- `render(blocks)`: Render the block manager UI with specific blocks.
