# Components API

The Components module manages the tree of components in the project.

## Methods

- `getWrapper()`: Returns the root component (wrapper).
- `getComponents()`: Returns the collection of components.
- `addComponent(component)`: Add a new component.
- `addType(id, definition)`: Add a new component type.
- `getType(id)`: Get component type definition.
- `addSymbol(component)`: Create a symbol from a component.
- `getSymbols()`: Get all symbols.
- `getSymbolInfo(component)`: Get details about a symbol.
