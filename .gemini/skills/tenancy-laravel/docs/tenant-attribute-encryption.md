                                <h1 id="encrypt">Tenant attribute encryption</h1>

To encrypt attributes on the Tenant model, store them in <a href="https://tenancyforlaravel.com/docs/v3/tenants/#custom-columns">custom columns</a> and cast the attributes to `'encrypted'`, or your custom encryption cast.



For example, we'll encrypt the tenant's database credentials – `tenancy_db_username` and `tenancy_db_password`. We need to create custom columns for these attributes, because by default, they are stored in the virtual `data` column.



<ul>
<li>Add custom columns to the tenants table (we recommend making the string size at least 512 characters, so the string is capable of containing the encrypted data, they also need to be `nullable` since they are filled after creation):</li>
</ul>

```
<code class="language-php">&lt;?php

use Illuminate\Database\Migrations\Migration;
use Illuminate\Database\Schema\Blueprint;
use Illuminate\Support\Facades\Schema;

class CreateTenantsTable extends Migration
{
    /**
     * Run the migrations.
     *
     * @return  void
     */
    public function up(): void
    {
        Schema::create('tenants', function (Blueprint $table) {
            $table-&gt;string('id')-&gt;primary();

            // Your custom columns
            $table-&gt;string('tenancy_db_username', 512)-&gt;nullable();
            $table-&gt;string('tenancy_db_password', 512)-&gt;nullable();

            $table-&gt;timestamps();
            $table-&gt;json('data')-&gt;nullable();
        });
    }
}
</code>
```


<ul>
<li>Define the custom columns on the Tenant model:</li>
</ul>

```
<code class="language-php">public static function getCustomColumns(): array
{
    return [
        'id',
        'tenancy_db_username',
        'tenancy_db_password',
    ];
}
</code>
```


<ul>
<li>Then define casts for the attributes on the model (using <a href="https://laravel.com/docs/9.x/eloquent-mutators#encrypted-casting">Laravel's encrypted casts</a>, or your custom casts):</li>
</ul>

```
<code class="language-php">protected $casts = [
    'tenancy_db_username' =&gt; 'encrypted',
    'tenancy_db_password' =&gt; 'encrypted',
];
</code>
```
